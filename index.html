Including expressions in strings

You can include JavaScript expressions in template literals, as well as just variables, and the results will be included in the results

Expressions in strings examples

          //Expressions in Strings//

const score = 9.6;
const highestScore = 10;
const output = ` The song, ${song} slaps. I give it ${
  (score / highestScore) * 100

Numbers vs Strings

What happens when we try to concatenate a string and a number? Let's try it in our console:

Numbers vs Strings Example

        //Numbers vs Strings
        
        const name_NvS = "Front ";
        const number_NvS = 242;
        console.log(name_NvS + number_NvS);

You might expect this to return an error, but it works just fine. How numbers should be displayed as strings is fairly well-defined, so the browser automaticlally converts the number to a string and concatenates the two strings.

If you have a numeric variable that you want to convert to a string, or a string variable that you want to convert to a number, you can use the following two constructs:

The Number() function converts anything passed to it into a number, if it can. Try the following:

    //Number() function//
    const myString = "123";
    const myNum = Number(myString);
    console.log(typeof myNum);
    const button_6 = document.querySelector("btn_6");

Conversely, the String() function converts its argument to a string.

          const myNum2 = "123";
